# ❤️ Love Calculator - Python Tkinter

A fun and interactive **Love Calculator GUI application** built with **Python Tkinter**.
This application calculates the **love compatibility percentage between two names** and displays a unique message based on the result.

The project includes a **beautiful graphical interface, background music using Pygame, and special love pairs for perfect matches**.

---

## 🚀 Features

* ❤️ Beautiful **Tkinter GUI interface**
* 🎵 **Background music** support using Pygame
* 💘 Calculates **love compatibility percentage**
* 💬 Displays **custom love messages**
* ⭐ Special love pairs always return **100%**
* 🎨 Attractive **pink themed UI**
* ⚡ Fast and lightweight Python application
* 🧑‍💻 Beginner friendly **Python GUI project**

---

## 🛠 Technologies Used

* Python
* Tkinter (GUI)
* Pygame (music)
* Random module
* Threading

## 📥 Installation

1. Clone this repository

```bash
git clone https://github.com/yourusername/love-calculator-tkinter.git
```

2. Navigate to the project folder

```bash
cd love-calculator-tkinter
```

3. Install required library

```bash
pip install pygame
```

4. Run the application

```bash
python love_calculator.py
```

---

## 🎯 How It Works

1. Enter two names.
2. Click **Calculate Love**.
3. The program calculates a compatibility score.
4. A love message is displayed based on the score.
